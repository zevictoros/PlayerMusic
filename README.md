# PlayerMusic

## Proposta

Algoritmo em linguagem C que emule as características de um player de músicas, sendo executado em modo texto, via prompt de comando.


## Operações

- Criar uma playlist das músicas.
- Armazenar o nome de cada música, o artista/banda e a duração da faixa.
- Armazenamento utilizando uma estrutura heterogênea de dados.
- Utilizar um menu para dá a opção do usuário escolher entre:

a. Criar playlist;

b. Inserir música na playlist;

c. Pesquisar uma determinada música;

d. Listar todas as músicas de uma determina playlist;

e. Deletar uma música(específica) de uma determinada playlist;

f. Deletar a playlist.
